# 303 Blog

## Setup

```
npm install
```

## Lint

```
npm run lint
```

## Test

* Update `.env.test` with your test db connection info
* Update scripts/test.sh with your test db name

```
npm run test
```

## Development

* Copy `.env.sample` to `.env`, update with your db connection information

```
npm run migrate
npm run seed
npm run dev
```

## Package Audit with snyk

```
npm run audit
```
